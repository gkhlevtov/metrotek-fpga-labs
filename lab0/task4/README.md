# Задание №4 CRC-16-ANSI

## Описание

В рамках лабораторной работы реализован модуль, вычисляющий контрольную сумму по алгоритму CRC-16-ANSI для последовательных данных. Модуль разработан в виде блок-диаграммы (crc_16_ansi.bdf) и использует D-триггеры, а также XOR для реализации алгоритма расчета циклического избыточного кода.

## Инструкция по запуску симуляции

1. Перейти в директорию `tb`:
   ```
   cd tb
   ```
2. Запустить ModelSim и выполнить команду:
   ```
   do run.do
   ```

## Проверка результатов работы модуля

Для проверки корректности работы модуля и функции, имитирующей подсчёт контрольной суммы через CRC-16-ANSI, использовался [ONLINE CRC BCH CALCULATOR - CODE GENERATOR](https://leventozturk.com/engineering/crc/).
Проверка проводилась при заданных параметрах:
- Polynomial:           `11000000000000101 Bin Xn to X0`
- Initialise:                `00000000000000000`
- Data width:            `32`
- Input data:             `Bin MSB bit/byte`
- Process Direction: `d[n] to d[0]`
- Output Format:      `o[n] to o[0] Bin`

На вход подавались конкретно заданные 32-битные наборы входных данных.
В ходе симуляции были получены следующие результаты:
1. Для `11111111111111111111111111111111`:
   * online calculator:   `1000000000101001`
   * crc_16_ansi_func: `1000000000101001`
   * crc_16_ansi:          `1000000000101001`
2. Для `00000000000000000000000000000000`:
   * online calculator:   `0000000000000000`
   * crc_16_ansi_func: `0000000000000000`
   * crc_16_ansi:          `0000000000000000`
3. Для `10101010101010101010101010101010`:
   * online calculator:   `1111111111100100`
   * crc_16_ansi_func: `1111111111100100`
   * crc_16_ansi:          `1111111111100100`
