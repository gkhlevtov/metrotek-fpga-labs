# Задание №4 CRC-16-ANSI

## Описание

В рамках лабораторной работы реализован модуль, вычисляющий контрольную сумму по алгоритму CRC-16-ANSI для последовательных данных. Модуль разработан в виде блок-диаграммы (crc_16_ansi.bdf) и использует D-триггеры, а также XOR для реализации алгоритма расчета циклического избыточного кода.

## Структура проекта

```
task3/
│── quartus_prj/               # Файлы проекта Quartus
│ ├── crc_16_ansi.qpf
│ ├── crc_16_ansi.qsf
│ └── crc_16_ansi.sdc
│── rtl/                       # Исходный код
│ ├── crc_16_ansi.bdf
│ └── crc_16_ansi.v
│── tb/                        # Тестбенч
│ ├── crc_16_ansi_tb.sv
│ └── run.do
└── README.md
```

## Инструкция по запуску симуляции

1. Перейти в директорию `tb`:
   ```
   cd tb
   ```
2. Запустить ModelSim и выполнить команду:
   ```
   do run.do
   ```
