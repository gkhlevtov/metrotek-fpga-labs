# Задание №2 Сортировка

## Описание

В рамках лабораторной работы реализован модуль `avalon_st_sort`, предназначенный для упорядочивания данных внутри пакетов. Архитектура модуля спроектирована с учетом совместимости по поведению и таймингам со спецификацией интерфейса Avalon Streaming (Avalon-ST), включая поддержку сигналов управления потоком (ready/valid) и границ пакета (SOP/EOP).
Параметры `DWIDTH` и `MAX_PKT_LEN` определяют разрядность данных и максимально допустимый размер пакета соответственно. Сортировка данных запускается автоматически по сигналу `endofpacket`, после чего упорядоченная последовательность выдается на выходной интерфейс.

## Инструкция по запуску симуляции

1. Перейти в директорию `tb`:
   ```
   cd tb
   ```
2. Запустить ModelSim и выполнить команду:
   ```
   do run.do
   ```
