# Задание №1 FIFO

## Описание

В рамках лабораторной работы реализован модуль fifo, предназначенный для организации буферизации данных в одном тактовом домене. Архитектура модуля спроектирована с учетом полной совместимости по поведению и таймингам со стандартным IP-ядром Intel scfifo (библиотека altera_mf).

Массив данных модуля синтезируется в аппаратные блоки памяти. Модуль функционирует в режиме Show-ahead (`SHOWAHEAD` = 1). Параметры `DWIDTH` и `AWIDTH` определяют разрядность данных и глубину очереди (2^AWIDTH слов) соответственно. Реализована поддержка пороговых флагов almost_empty_o и almost_full_o, значения для которых задаются через параметры `ALMOST_EMPTY_VALUE` и `ALMOST_FULL_VALUE`.
