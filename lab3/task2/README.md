# Лабораторная №3

# Задание №2 Конвертер ширины поточного интерфейса

## Описание

В данной лабораторной работе был верифицирован модуль конвертер ширины `ast_width_extender`, который преобразует ширину данных Avalon‑ST из меньшей в кратную ей большую.  
Тестовое окружение реализовано с использованием объектно-ориентированного подхода: классы генератора, драйвера, монитора и скорборда объединены в тестовой среде.


## Структура проекта

```
task2/
├── rtl/                        # Исходный код верифицируемого модуля
│ └── ast_width_extender.sv
├── tb/                         # Тестовое окружение
│ ├── ast_in_if.sv              # Входной Avalon‑ST интерфейс
│ ├── ast_out_if.sv             # Выходной Avalon‑ST интерфейс
│ ├── ast_transaction.sv        # Описание транзакции Avalon‑ST
│ ├── ast_width_extender_pkg.sv # Пакет с описанием классов
│ ├── ast_width_extender_tb.sv  # Верхний уровень тестбенча
│ ├── driver.sv                 # Драйвер для подачи сигналов на вход DUT
│ ├── environment.sv            # Контейнер для сборки всех компонентов
│ ├── generator.sv              # Генератор тестовых сценариев и последовательностей
│ ├── monitor.sv                # Монитор для захвата выходных данных DUT
│ ├── run.do                    # Скрипт запуска симуляции
│ └── scoreboard.sv             # Эталонная модель и проверка данных
├── README.md
└── report.md                   # Отчет о найденных багах
```

## Инструкция по запуску симуляции

1. Перейти в директорию `tb`:
   ```
   cd tb
   ```
2. Запустить ModelSim и выполнить команду:
   ```
   do run.do
   ```

## В ходе работы использовались

- **Quartus Prime Lite 18.1**
