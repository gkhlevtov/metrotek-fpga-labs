# Задание №6 Светофор

## Описание

В рамках лабораторной работы реализован модуль traffic_lights, предназначенный для управления светофором с поддержкой стандартного цикла работы, режима неуправляемого перехода и динамической настройки временных параметров.

Модуль принимает тактовый сигнал clk_i с частотой 2 кГц, синхронный сигнал сброса srst_i и управляющую шину команд cmd_i, представленную сигналами cmd_type_i, cmd_valid_i и cmd_data_i. Управление осуществляется с помощью конечного автомата, который последовательно формирует состояния: "красный", "красный + жёлтый", "зелёный", "зелёный мигающий", "жёлтый" и возврат в "красный". При включении или сбросе модуль переходит в состояние "красный".

Длительности горения красного, жёлтого и зелёного сигналов задаются в миллисекундах через команды управления и могут изменяться только в режиме неуправляемого перехода. Параметры `BLINK_HALF_PERIOD_MS`, `BLINK_GREEN_TIME_TICK` и `RED_YELLOW_MS` определяют полупериод мигания, продолжительность мигающего зелёного сигнала и время состояния "красный + жёлтый" соответственно. Состояние светофора отображается на выходах red_o, yellow_o и green_o.

## Инструкция по запуску симуляции

1. Перейти в директорию `tb`:
   ```
   cd tb
   ```
2. Запустить ModelSim и выполнить команду:
   ```
   do run.do
   ```
