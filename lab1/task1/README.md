# Задание №1 Сериализатор

## Описание

В рамках лабораторной работы реализован модуль serializer, выполняющий последовательную передачу параллельных данных фиксированной ширины (16 бит).
Модуль принимает входные данные, управляющий параметр количества валидных бит, а также сигнал валидности входа. На основании этих сигналов сериализатор формирует последовательный поток данных и соответствующий сигнал валидности на выходе.

## Инструкция по запуску симуляции

1. Перейти в директорию `tb`:
   ```
   cd tb
   ```
2. Запустить ModelSim и выполнить команду:
   ```
   do run.do
   ```
