# Задание №4 Количество единиц

## Описание

В рамках лабораторной работы реализован модуль bit_population_counter, предназначенный для подсчёта количества единиц во входном двоичном слове заданной разрядности, определяемой параметром (`WIDTH`).

Модуль принимает входные данные data_i и сигнал их валидности data_val_i. После обработки входного слова на выход data_o выдаётся количество установленных бит, а сигнал data_val_o подтверждает валидность результата. Задержка обработки составляет не менее одного такта и зависит от разрядности входных данных.

## Инструкция по запуску симуляции

1. Перейти в директорию `tb`:
   ```
   cd tb
   ```
2. Запустить ModelSim и выполнить команду:
   ```
   do run.do
   ```
