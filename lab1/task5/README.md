# Задание №5 Антидребезг

## Описание

В рамках лабораторной работы реализован модуль debouncer, предназначенный для подавления дребезга механической кнопки и формирования одиночного строба при устойчивом нажатии.

Модуль принимает тактовый сигнал clk_i и асинхронный вход кнопки key_i, активный в низком уровне. Длительность подавления дребезга задаётся параметром `GLITCH_TIME_NS` в наносекундах и пересчитывается во внутреннее количество тактов на основе частоты тактирования, определяемой параметром `CLK_FREQ_MHZ`.

## Инструкция по запуску симуляции

1. Перейти в директорию `tb`:
   ```
   cd tb
   ```
2. Запустить ModelSim и выполнить команду:
   ```
   do run.do
   ```

## Проверка результатов работы модуля

Проверка корректности работы модуля проводилась при параметрах тестбенча:

- N = 50 (const)
- WAIT_TIME = 5 (const)
- T = 10, 20, 40

Использованные при тестировании наборы параметров модуля:
1. CLK_FREQ_MHZ   = 100
   GLITCH_TIME_NS = 10
2. CLK_FREQ_MHZ   = 150
   GLITCH_TIME_NS = 20
3. CLK_FREQ_MHZ   = 200
   GLITCH_TIME_NS = 20
4. CLK_FREQ_MHZ   = 150
   GLITCH_TIME_NS = 40
