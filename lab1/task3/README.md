# Задание №3 Крайние единицы

## Описание

В рамках лабораторной работы реализован модуль priority_encoder, предназначенный для поиска крайних единиц во входном слове заданной разрядности, определяемой параметром (`WIDTH`).

Модуль принимает валидные входные данные и формирует два выходных вектора: с сохранённой только самой левой и только самой правой единицей входного числа. При отсутствии единиц на входе оба выхода принимают нулевое значение. Сигнал data_val_o подтверждает валидность сформированных выходных данных.

1. Перейти в директорию `tb`:
   ```

   ```
